# Deep Audience Analysis Premium

Elite marketing analysis of target audience psychology, motivations, and behavior.

## When to Use

- Building detailed customer personas
- Preparing for sales copy, landing pages, ad campaigns
- Understanding why customers buy or don't buy
- Developing positioning and USP
- Working with objections
- Content strategy based on audience psychology

## Workflow

1. **Collect context** — gather product, market, audience, competitor info
2. **Check data sufficiency** — if missing key info, ask clarifying questions first
3. **Run full analysis** — follow the 21-section framework from the master prompt
4. **Deliver actionable output** — marketing-ready insights

## Output Format

The analysis follows a strict 21-section structure:

1. Краткое резюме
2. Профиль идеального клиента
3. Психографика и отношение к жизни
4. Основная проблема
5. JTBD (Jobs To Be Done)
6. Топ-5 эмоций вокруг проблемы
7. Топ-5 самых сильных страхов
8. Влияние проблемы на отношения
9. Болезненные фразы
10. Что клиент уже пробовал
11. Чего клиент не хочет делать
12. Идеальная трансформация
13. Пост-трансформационные цитаты
14. Мифы, убеждения и заблуждения
15. Кого клиент винит
16. Где клиент застрял
17. Что клиенту нужно отпустить
18. Топ-5 возражений против покупки
19. Ключевые триггеры покупки
20. Маркетинговые выводы
21. Гипотезы и степень уверенности

## Language

Default: **Russian**. Switch to English only when the source material is English or the user explicitly requests it.

## Rules

- Never invent facts, research data, or testimonials
- Mark hypotheses clearly as hypotheses
- If data is insufficient, ask clarifying questions before analyzing
- Analyze each segment separately when multiple segments are requested
- Use the customer's own language — conversational, authentic, specific
