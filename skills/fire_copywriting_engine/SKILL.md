# Fire Copywriting Engine

Transform dry facts into fire sales copy that converts.

## When to Use

- Rewriting weak marketing text into persuasive copy
- Creating sales pages, landing pages, ads, email campaigns
- Improving existing copy that doesn't convert
- Any text that needs to sell, persuade, or drive action

## Workflow

1. **Scan the source material** — extract key facts, offers, audience
2. **Psychological profiling** (Sugarman/Bencivenga) — identify deep pains, fears, desires of the target audience
3. **Structure with AIDA** (Ogilvy) — Attention → Interest → Desire → Action
4. **Write the fire copy** — apply all algorithms from the master prompt
5. **Strike with CTA** (Carlton) — uncompromising call to action

## Core Rules

- **Show, Don't Tell** — never write abstractly. Use vivid visual scenes the reader can see, smell, hear
- **"Свой парень" style** (Ben Settle) — conversational, light language. Zero bureaucracy
- **Fact foundation** (Kennedy/Bly) — back promises with specific data and social proof
- **Cinematographic text** — every sentence is a step closer to the purchase button
- **Empathy first** — fill the text with understanding of the reader's pain

## Output Format

When given source text, produce:
1. **Огненный продающий текст** — the fire copy itself
2. **Пояснение принятых решений** — brief rationale for key choices (optional, on request)
